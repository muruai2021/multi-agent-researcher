## 6 类失败模式分类

| 类别 | 名称 | 触发条件 | 默认处理 |
|------|------|----------|----------|
| **L1** | 调度失败 | 调度超时 2 分钟 | 重试 1 次 → 跳过 |
| **L2** | 数据缺失 | Scout 数据包为空/不完整 | 标记缺口 + 旁证 |
| **L3** | 输出不合格 | Reviewer 标 Critical / High > 2 | Writer 修订 ≤ 3 轮 |
| **L4** | 资源耗尽 | Token / 速率超限 | 保存 CP + 询问用户 |
| **L5** | 用户交互失败 | 追问 3 次未明确 | 提供 3 路径让用户选 |
| **L6** | 协议违反 | Agent 未按协议执行 | 主编干预 + 记录 |
